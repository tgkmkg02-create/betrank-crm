1. 制度概要

スタッフは 月単位 で 2 つのプランから選べます。前月 20 日までに翌月分を申請してください。
自分のスタイル・その月の見込みに合わせて選択することで、納得感のある給与体系を実現します。

Pattern A
歩合振り切り型 (攻め)

基本給を抑え、歩合・ボーナスを厚くする。
高パフォーマンス時に大きく稼げる。
向いている人: 顧客対応に自信がある / 月によって稼ぎを大きくしたい / 評価軸で勝負したい

Pattern B
現状維持型 (守り)

基本給を上げ、歩合は控えめ。
低パフォーマンスでも収入が安定する。
向いている人: 安定収入を優先したい / 新人 / 繁忙月以外は手堅くやりたい

6. 運用ルール

📋 推奨運用ルール
  • 選択は 毎月 20 日まで に翌月分を申請
  • 初月は全員 Pattern B (固定型) から開始 (リスク低減)
  • 連続 3 ヶ月以上 Pattern A を選択した場合、達成率に応じて評価
  • Pattern A 選択時の C 評価が 3 ヶ月続いた場合、強制的に B へ戻す (保護措置)
  • 新人 (入社 6 ヶ月以内) は Pattern B 限定

7. クイックサマリー (説明用 1 ページ要約)

制度の核

  • スタッフが 自分で月単位 でプランを選べる
  • 「攻め (A)」か「守り (B)」かは自己判断
  • テレベット誘導は 別枠で全員対象。プラン選択に影響しない

選び方の目安

こんな月はおすすめ理由
普段通り or やや控えめ Pattern B 基本給が高いので底堅い
S 評価を狙える / 紹介・リピート見込みあり Pattern A 歩合とボーナスが大きく稼げる
新人 / 慣れていない月 Pattern B 強制 (保護措置)
テレベット誘導が多い月 どちらでも OK 別枠ボーナスなので影響なし

損益分岐点 (覚えておく数字)

  • 韓国現地: エース級 (S 評価 + 複数評価軸で活躍) を超えたら A 検討
  • 日本チーム: 月 8 件 × 200 万円プラン (売上 1,600 万) がライン

変更可能タイミング

  • 毎月 20 日まで に翌月分を申請
  • 申請なし → 前月と同じプランを継続
