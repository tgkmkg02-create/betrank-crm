給与プランA/B制度設計書

ベットランクツアーズ — 現地スタッフ・日本チーム向け新給与制度
制度概要: スタッフは月単位で2つのプラン（A: 歩合振り切り型 / B: 現状維持型）から選択可能。 前月25日までに翌月分を申請。テレベット誘導ボーナスは別枠で全員対象（A/B共通）。
適用対象: 韓国現地スタッフ（チョイ・ケビン・チェリン）、日本チーム（長岡きみ・長岡ゆみ）
非適用: アバタースタッフ（たく・むらまつ・あやか）は現行のポイント制を維持

① 現地スタッフ（チョイ・ケビン・チェリン）

基本給は現在の月給を維持し、Pattern A/Bは歩合の大きさだけで差をつける設計。

Pattern A — 歩合振り切り型 攻め
  • 基本給: 現給 × 0.75（−25%）
  • ランクボーナス（×2倍）: S ¥60,000 / A ¥36,000 / B ¥16,000 / C ¥0
  • 顧客アンケート個人名評価: +¥3,000/回
  • リピート獲得: +¥5,000/件
  • 新規顧客紹介: +¥10,000/件
Pattern B — 現状維持型 守り
  • 基本給: 現給そのまま
  • ランクボーナス（現行）: S ¥30,000 / A ¥18,000 / B ¥8,000 / C ¥0
  • 評価ボーナス: なし
★ テレベット誘導 別枠ボーナス（A/B共通・全員対象）

テレベット誘導は本人のパターン選択に関係なく、誘導した件数に応じて月次で支給。

③ 運用ルール

選択ルール

  • 選択は毎月25日までに翌月分を申請
  • 初月は全員 Pattern B（現状維持型） から開始（リスク低減）
  • 新人（入社6ヶ月以内）は Pattern B 限定
  • 選択は月単位で変更可能（前月25日までの申請で翌月から）

保護措置

  • 連続3ヶ月以上 Pattern A を選択した場合、達成率に応じて評価
  • Pattern A選択時の C評価が3ヶ月続いた場合、強制的にBへ戻す
  • Pattern B はいつでも選択可能・現給は保証

テレベット誘導ボーナス

  • パターンA/Bの選択に関係なく、誘導件数に応じて全員に支給
  • 1件 ¥3,000 + 5件達成 ¥10,000 + 10件達成 ¥30,000（達成ボーナスは段階累積）
  • 例: 12件 → ¥3,000×12 + ¥30,000 = ¥66,000

評価項目（Pattern A選択時のみ加算）

  • ランクボーナス: 月次4軸評価 + 3名評価者で決定（営業貢献・顧客対応・勤怠・チーム）
  • 顧客アンケート個人名評価: アンケートで個人名が記載された回数
  • リピート獲得: 既存顧客の再訪を獲得した件数
  • 新規顧客紹介: 新規顧客を開拓・紹介した件数

💡 設計のメリット

  • 低パフォーマンス時の人件費抑制: 全員Aで成果なしなら本体給与は▲6.7%減
  • 高パフォーマンス時のリターン正比例: エース級は会社の売上も大きく伸びる前提
  • スタッフのモチベーション設計: 「選んだ責任」がパフォーマンスを引き出す
  • テレベット別枠: 本来業務（カジノ顧客対応）と独立した評価が可能
